D点在B点右侧时

需要自己画出这种情况的示意图.

同第一种情况一样, 我们首先找出90度角,然后求解长度.

  1. 猜想:

    $只有\angle BDE 可能等于90 ^o$

  2. 类似第一种情况, 可以证明 (这个请自行证明, 之后可以看提示2进行验证):

    $\angle EBD = 60^o, \angle DEB \leq 60 < 90^o$

  3. 假设$\angle BDE = 90 ^o$, 可以求出长度(这个请自行证明, 之后可以看提示3进行验证)

    $\therefore \triangle ACD是30^o, 60^o, 90^o 的特殊直角三角形, \ \therefore BD = AB = AC = 4 $

考点

  1. 分情况讨论的思想；
  2. 旋转全等；
  3. 正三角形；
  4. 特殊三角形的熟练掌握。
